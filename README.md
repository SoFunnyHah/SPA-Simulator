# SPA-Simulator

How to use:

1. Git clone repositorie
<br/>
Terminal vs code:
<br/>
2. cd server 
<br/>
3. npm i
<br/>
4. Add new terminal
<br/>
5. cd client
<br/>
6. npm i
<br/>
7. Create .env on client and server parts
<br/>
Simpleenv.txt will help to see constant names
<br/>
Example of .env result:
<br/>
Client part:
<br/>
REACT_APP_BASEURL=http://localhost:3001
<br/>
Server part:
<br/>
PORT=3001
<br/>
8. npm start on both terminals
